# 🦭Обработка текста🦭

Данный проект посвящен решению задач NLP (обработки естественного языка) с использованием метода **Bag of Words** на языке Python с использованием библиотеки TensorFlow/Keras.

## 📝 Описание проекта

В репозитории представлены два основных направления исследования:
1.  **Классификация авторов:** Определение авторства текста среди нескольких писателей. Основной фокус сделан на подборе оптимальных гиперпараметров (размер словаря, ширина окна, шаг).
2.  **Анализ тональности отзывов:** Определение позитивного и негативного отношения в отзывах об автомобилях Tesla с достижением точности 85-90%.

## 🛠 Технологический стек
* **Python 3.x**
* **TensorFlow / Keras** — построение и обучение нейросетей.
* **NumPy / Pandas** — обработка и структурирование данных.
* **Matplotlib** — визуализация процесса обучения (Loss/Accuracy).
* **Scikit-learn / Tokenizer** — векторизация текстов.

## 🚀 Основные этапы работы

### 1. Предобработка данных
Для работы с текстами использовался метод "скользящего окна", позволяющий нарезать длинные произведения на обучающие выборки заданного размера.
* Тексты переведены в токены и векторизованы через `sequences_to_matrix`.
* Для экономии оперативной памяти данные приведены к типу `uint8`.

### 2. Эксперименты с гиперпараметрами (Lite)
Было обучено **9 различных моделей** с варьированием следующих параметров:
* Размер словаря: от 10 000 до 20 000 слов.
* Ширина окна: от 1000 до 2000 слов.
* Шаг окна: от 100 до 500 слов.
